# Screenshot Checklist

Use this checklist when preparing marketplace screenshots.

## Required Screenshots To Capture

1. Cura main window with a sliced model loaded
2. `Extensions -> S1 Pro Auto Thumbnail` menu visible
3. Plugin settings window open
4. Settings window showing thumbnail options
5. Settings window showing leveling options
6. Exported `.gcode` opened in a text editor with the inserted `; jpg begin ...` block visible
7. Optional: printer screen showing the generated thumbnail preview

## Capture Guidance

- Use a clean Cura workspace with one simple model loaded
- Avoid clutter from unrelated plugins or dialogs
- Keep the active printer profile visible if it is relevant
- Use readable zoom so text in the settings window is legible
- Do not crop away the Cura frame entirely; it should be obvious the plugin runs inside Cura
- If showing the printer preview, use a sharp photo with minimal glare

## Suggested Screenshot Order For Listing

1. Main value shot: settings window inside Cura
2. Automatic export workflow
3. Resulting G-code thumbnail block
4. Real printer preview photo
